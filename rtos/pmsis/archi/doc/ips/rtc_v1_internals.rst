.. table::

    |Bit #|R/W|  Name  |Reset|                                                                Description                                                                |
    +=====+===+========+=====+===========================================================================================================================================+

    |    8|R/W|SOFT_RST|0x0  |Soft reset command: b0: no reset; b1: reset the calendar, alarm, countdown and clock generation features and associated registers (CALENDAR|
